1Ô∏è‚É£ Understanding the CSS Box Model

Every HTML element is a box consisting of:

  • Content (text, images, etc.)
  • Padding (space inside the box, around content)
  • Border (surrounds padding and content)
  • Margin (space outside the border, separates from other elements)

3Ô∏è‚É£ Box Sizing

  • Default is box-sizing: content-box; (width and height apply to content only)
  • box-sizing: border-box; includes padding and border in width and height calculations
  • Using border-box often simplifies layout

4Ô∏è‚É£ Display Property

  • block: Elements take full width, start on a new line (e.g., <div>, <p>)
  • inline: Elements flow within a line, width/height ignored (e.g., <span>, <a>)
  • inline-block: Like inline but respects width/height
  • none: Hide element
