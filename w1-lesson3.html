💻 Lesson 3: Getting Started with HTML

🕒 Duration:

~45–60 minutes (Third segment of Week 1 — includes explanation + hands-on practice)

🎯 Learning Objectives:

  • Understand what HTML is and what it's used for
  • Recognize the role of HTML in building websites
  • Set up a simple HTML file using a text editor
  • Write and view a basic HTML page in a browser

🧩 Topics Covered:

📌 3.1 What is HTML?

  • HTML stands for HyperText Markup Language
  • It defines the structure and content of web pages
  • HTML uses tags (also called elements) to describe parts of a webpage
  • It's the foundation of all websites
📌 HyperText = text with links (hyperlinks)
📌 Markup Language = uses tags to “mark up” text to give it structure

⚙️ 3.2 What Does HTML Do?

HTML tells the browser what to display, such as:

  • Headings
  • Paragraphs
  • Links
  • Images
  • Lists
  • Tables
  • Forms

It does not control appearance — that’s what CSS is for.

🛠️ 3.3 Setting Up: First HTML Page

✔ Tools Needed:

  • A text editor (e.g., Visual Studio Code)
  • A browser (e.g., Chrome, Firefox)

🗂 Folder & File Setup:

Create a folder named my-first-website, and inside it, create a file called:

index.html

✍ First HTML Example:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>

🔍 3.4 Explanation of Structure:

Tag Purpose
<!DOCTYPE html> Declares the document is HTML5
<html> Root element of the page
<head> Metadata (not displayed in browser window)
<title> Title of the page (seen in browser tab)
<body> Visible content of the webpage
<h1> Heading (largest)
<p> Paragraph

🧪 Activities / Practice

✅ Activity 1: Code-Along

Students type the example HTML code in their own index.html, save it, and open it in their browser.

✅ Activity 2: Modify Your Page

Ask students to:

  • Change the <title>
  • Add another heading (<h2>)
  • Add a second paragraph
  • Add their name to the page

📝 Key Takeaways:

  • HTML is the language used to structure content on the web
  • Every web page starts with basic tags like <html>, <head>, and <body>
  • You can write HTML in any text editor and view it in a browser

📚 Optional Extension:

Introduce online playgrounds like:

  • CodePen
  • JSFiddle
  • JSBin

This helps students who can’t install editors or want to test on the go.
