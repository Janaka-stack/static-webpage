ğŸ–¥ï¸ 1.3 How Websites Work (Clientâ€“Server Model)

Step-by-step breakdown:

  1. User types a URL into the browser.
  2. The browser sends a request to a DNS server to get the IP address.
  3. The browser sends an HTTP request to the web server.
  4. The server processes the request and sends back the requested files (HTML, CSS, images, etc.).
  5. The browser receives the files and renders the web page.
