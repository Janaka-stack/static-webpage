4. Basic Structure of an HTML Document

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>

Tags explained: <!DOCTYPE html>, <html>, <head>, <body>, <title>
